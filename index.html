Quick Start Installation

The SoulBits AI Docker image includes the full neural inference stack, REST API gateway, and WebSocket event stream. Requires Docker 24+ and 8GB RAM minimum.

soulbits — config.yaml
# soulbits.config.yaml
api:
  port: 8080
  auth: bearer
neural:
  model: soulbits-v2.4
  context_window: 131072
  temperature: 0.7
security:
  encryption: aes-256-gcm
  key_rotation: 24h

Environment Configuration

Mount your config file with -v ./config.yaml:/app/config.yaml or set environment variables SOULBITS_API_KEY, SOULBITS_MODEL, and SOULBITS_PORT for containerized deployments.

soulbits — api test
# Send a chat completion request
$ curl -X POST http://localhost:8080/v1/chat \
  -H "Authorization: Bearer $SOULBITS_API_KEY" \
  -d '{"message": "Hello, SoulBits"}'

REST API Endpoints

The v1 API supports /chat, /voice, /sync, and /companions endpoints. WebSocket streaming is available at ws://localhost:8080/v1/stream for real-time token generation.
